# 덱 편성 시스템 (DeckSetting)

## 구현 상태: Pawn 탭 완성, Card 탭 기본 구현 완료

---

## 핵심 파일

| 파일 | 역할 |
|------|------|
| `Assets/Scripts/UI/DeckSetting/DeckSettingPage.cs` | 덱 편성 페이지 — Pawn/Card 탭 전환 |
| `Assets/Scripts/UI/DeckSetting/DeckSetting_Pawn.cs` | Pawn 편성 서브 페이지 — 보유 목록 표시, 슬롯 편성 |
| `Assets/Scripts/UI/DeckSetting/DeckPawnItem.cs` | 보유 Pawn 아이템 — 드래그 & 우클릭 편성 |
| `Assets/Scripts/UI/DeckSetting/DeckPawnSlot.cs` | 편성 슬롯 — PawnIcon 표시, 우클릭 해제 |
| `Assets/Scripts/UI/DeckSetting/DeckSetting_Card.cs` | Card 서브 페이지 — 보유 카드 목록 그리드 표시 |
| `Assets/Scripts/UI/DeckSetting/DeckCardItem.cs` | 보유 카드 아이템 — CardIcon 표시 |
| `Assets/Scripts/Logic/Manager/DeckManager.cs` | 편성 슬롯(`_deckPawnSlots`) + 카드 덱(`_deckCardList`) 관리 |

---

## DeckManager Pawn 관련 API

| 메서드/프로퍼티 | 설명 |
|----------------|------|
| `OwnPawns` (IReadOnlyList) | 보유 전체 Pawn 목록 |
| `DeckPawns` (IEnumerable) | 편성된 Pawn 목록 (null 슬롯 제외, Stage용) |
| `OpenSlotCount` (int, 초기값 2) | 열린 슬롯 수 |
| `ExpandSlot()` | 슬롯 1개 확장 (최대 5), 성공 시 true |
| `AddPawn(DPawn)` | 영입한 Pawn을 보유 목록에 추가 |
| `AssignPawnToDeck(DPawn, int)` | 슬롯에 Pawn 편성 |
| `UnassignPawnFromDeck(int)` | 슬롯 편성 해제 |
| `FindEmptySlotIndex()` | 첫 번째 빈 슬롯 인덱스 반환 (-1: 없음) |
| `GetDeckPawn(int)` | 슬롯 인덱스로 편성된 Pawn 조회 |

```csharp
// 내부 구조
private readonly List<DPawn> _ownPawnList;   // 보유 전체
private DPawn[] _deckPawnSlots = new DPawn[5]; // 편성 슬롯
private int _openSlotCount = 2;               // 열린 슬롯 수 (추후 확장 가능)
```

## Pawn 편성 흐름

```
DeckSettingPage.OnOpened()
    └── DeckSetting_Pawn.OnOpened()
            ├── InitSlots()        — 슬롯 잠금 여부 설정, 기존 편성 표시
            └── RefreshOwnPawns()  — 보유 Pawn → DeckPawnItem 생성

DeckPawnItem 드래그 → DeckSetting_Pawn.OnItemDragEnd()
    └── GetSlotAtPosition() 로 슬롯 탐색
    └── DeckManager.AssignPawnToDeck() + DeckPawnSlot.SetData()

DeckPawnItem 우클릭 → DeckSetting_Pawn.OnItemRightClick()
    └── DeckManager.FindEmptySlotIndex()
    └── DeckManager.AssignPawnToDeck() + DeckPawnSlot.SetData()

DeckPawnSlot 우클릭 → DeckSetting_Pawn.OnSlotRightClick()
    └── DeckManager.UnassignPawnFromDeck() + DeckPawnSlot.Clear()
```

---

## DeckPawnItem

- **상속**: `MonoBehaviourEx`, `IPointerDownHandler`, `IDragHandler`, `IPointerUpHandler`, `IPointerClickHandler`
- **필드**: `Spawn _spawnIcon` → `PawnIcon` 표시
- **이벤트**:
  ```csharp
  event Action<DeckPawnItem, PointerEventData> OnDragBegin
  event Action<DeckPawnItem, PointerEventData> OnDragging
  event Action<DeckPawnItem, PointerEventData> OnDragEnd
  event Action<DeckPawnItem> OnRightClick
  ```
- 드래그 중 위치 이동, 드래그 종료 후 원위치 복귀
- 우클릭은 `PointerEventData.InputButton.Right` 판정

---

## DeckPawnSlot

- **상속**: `MonoBehaviourEx`, `IPointerClickHandler`
- **필드**: `_goEmpty` (빈 슬롯 표시), `_goLock` (잠금 표시), `Spawn _spawnIcon`
- **초기화**: `Init(slotIndex, isLocked)` — 인덱스 기록, 잠금/빈 슬롯 표시 설정
- **편성 표시**: `SetData(DPawn)` — `_goEmpty` 숨김, PawnIcon 표시
- **해제**: `Clear()` — `_goEmpty` 표시
- 우클릭 시 `OnRightClick` 이벤트 발생 (잠금/빈 슬롯 제외)
- 잠긴 슬롯 좌클릭 시 `OnLockedClick` 이벤트 발생 → 슬롯 업그레이드 팝업

---

## 슬롯 잠금 정책

| 슬롯 | 초기 상태 | 조건 |
|------|----------|------|
| 0, 1 | 열림 | 기본 제공 |
| 2, 3, 4 | 잠김 (`_goLock` 활성) | 추후 업그레이드로 개방 예정 |

- `DeckManager.OpenSlotCount` 값을 늘리면 추가 슬롯 개방
- 잠긴 슬롯은 드래그 드롭 불가 (`AssignPawnToDeck` 내부에서 검증)
- 잠긴 슬롯 좌클릭 → `NoticePopup`으로 확인 → 100골드 소모 → `DeckManager.ExpandSlot()` → UI 갱신

## DeckManager Card 관련 API

| 메서드/프로퍼티 | 설명 |
|----------------|------|
| `DeckCards` (IReadOnlyList) | 보유 전체 카드 목록 (읽기 전용) |
| `AddCard(DCard)` | 카드를 덱에 추가 |

```csharp
// 내부 구조
private readonly List<DCard> _deckCardList;  // 보유 카드 전체
public IReadOnlyList<DCard> DeckCards => _deckCardList;
```

---

## Card 탭 흐름

```
DeckSettingPage.OnClickCardTab()
    └── DeckSetting_Card.OnOpened()
            └── RefreshCardList()  — DeckCards → DeckCardItem 생성 (5열 그리드)

DeckCardItem.SetData(DCard)
    └── _spawnIcon 내부 CardIcon.SetData(CardData)
```

- 수직 스크롤, 한 줄에 5장 표시
- PawnManagePage 장비 그리드와 동일한 수동 RectTransform 배치 패턴 사용
- Content 높이를 행 수에 따라 동적 조정

---

## DeckCardItem

- **상속**: `MonoBehaviourEx`
- **필드**: `Spawn _spawnIcon` → `CardIcon` 표시
- **메서드**: `SetData(DCard)` — CardIcon에 카드 데이터 세팅
- 현재 단순 표시 전용 (드래그/클릭 이벤트 없음)
