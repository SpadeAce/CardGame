# 씬 흐름

## 구현 상태: 완성

---

## 씬 전환 흐름

```
BootScene
    │ DataManager.LoadAll()
    │ DeckManager.InitTestData()
    └──▶ TitleScene
              │ [New Game / Load Game 버튼]
              └──▶ LobbyScene
                        │ [Start Battle 버튼]
                        └──▶ StageScene
                                  │ [BattleResultPopup - 승리]
                                  │  PlayerManager.IncrementLevel()
                                  ├──▶ LobbyScene
                                  │ [BattleResultPopup - 패배]
                                  └──▶ TitleScene (Exit)
```

---

## 씬별 상세

### BootScene
- **스크립트**: `Assets/Scripts/Logic/Scene/BootScript.cs`
- **역할**: 게임 최초 진입점. 모든 데이터 로드 후 TitleScene으로 이동.
- **OnEnterScene() 동작**:
  1. `DataManager.Instance.LoadAll()` — Protobuf `.bytes` 파일 전체 로드
  2. `DeckManager.Instance.InitTestData()` — 카드 5종류 × 3장 테스트 덱 세팅
  3. `SceneController.Instance.ChangeScene("TitleScene")`

---

### TitleScene
- **스크립트**: `Assets/Scripts/Logic/Scene/TitleScript.cs`
- **UI**: `TitlePage`
- **버튼**:
  | 버튼 | 동작 |
  |------|------|
  | New Game | LobbyScene 전환 |
  | Load Game | LobbyScene 전환 (현재 Save/Load 미구현) |
  | Option | OptionPopup 열기 |
  | Exit | 게임 종료 |

---

### LobbyScene
- **스크립트**: `Assets/Scripts/Logic/Scene/LobbyScript.cs`
- **UI**: `LobbyPage`
- **버튼**:
  | 버튼 | 동작 | 구현 상태 |
  |------|------|---------|
  | Start Battle | StageScene 전환 (DeckPawns 없으면 무시) | ✅ 완성 |
  | Deck Setting | DeckSettingPage 오픈 | ✅ 완성 (Pawn 탭, Card 탭 미구현) |
  | Pawn Manage | PawnManagePage 오픈 | ✅ 완성 |
  | Shop | ShopPage 오픈 | ✅ 완성 |
  | Recruit | RecruitPage 오픈 | ✅ 완성 |
  | Exit | TitleScene 전환 | ✅ 완성 |

---

### StageScene
- **스크립트**: `Assets/Scripts/Logic/Scene/StageScript.cs`
- **UI**: `StagePage`
- **OnEnterScene() 동작**:
  1. `UIManager.OpenView<StagePage>()` — 게임 UI 표시
  2. `StageManager.LoadStage(ResolveStagePreset())` — 난이도 레벨에 맞는 TileMapPreset으로 맵 생성 및 Actor 스폰
     - `PlayerManager.DifficultyLevel` → `DataManager.StagePreset.GetByLevel()` → `PresetPath` 랜덤 선택 → `Resources.Load<TileMapPreset>()`
     - 로드 실패 시 Inspector의 `_stagePreset` 폴백
  3. `DeckManager.InitStage()` — 카드풀 섞기 + 초기 핸드 5장 드로우
  4. `CameraController` 카메라 위치 맵 중앙으로 이동
  5. `TurnManager.StartGame()` — Pawn 턴 시작
- **OnExitScene() 동작**:
  - `TileManager.Instance.ClearAllTiles()` — 타일맵 전체 제거
- **Update()**: 마우스 좌클릭 → `HandleTileClick()` (카드 드래그 중이면 무시)

## SceneController

- **파일**: `Assets/Scripts/Logic/Scene/Core/SceneController.cs`
- **패턴**: `Singleton<SceneController>` (순수 C# 클래스)
- **핵심 메서드**:
  ```
  ChangeScene(string sceneName)
      → UIManager.CloseAllView()
      → SceneManager.LoadScene(sceneName)

  OnChangeScene(Scene old, Scene current) [이벤트 콜백]
      → 씬 내 SceneBase 컴포넌트 검색
      → SceneBase.OnEnterScene() 호출
  ```

---

## SceneBase

- **파일**: `Assets/Scripts/Logic/Scene/SceneBase.cs`
- **패턴**: Abstract MonoBehaviour
- **추상 멤버**:
  ```
  abstract string SceneName
  abstract void OnEnterScene()
  abstract void OnExitScene()
  ```
