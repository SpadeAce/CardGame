# 영입 시스템 (Recruit)

## 구현 상태: 완성 (재화 차감, LobbyManager 데이터 분리 완료)

---

## 핵심 파일

| 파일 | 역할 |
|------|------|
| `Assets/Scripts/UI/Recruit/RecruitPage.cs` | 영입 페이지 — UI 렌더링, 이벤트 처리 |
| `Assets/Scripts/UI/Recruit/RecruitItem.cs` | 영입 아이템 — 개별 Pawn 표시, 영입 버튼 처리 |
| `Assets/Scripts/Logic/Manager/LobbyManager.cs` | 영입 후보 목록 보관 (페이지 독립) |
| `Assets/Scripts/Logic/Manager/DeckManager.cs` | 영입된 Pawn을 덱에 추가 (`AddPawn`), 골드 관리 |
| `Assets/Scripts/Logic/Manager/TableData/PawnTable.cs` | 전체 PawnData 목록 조회 (`GetAll`) |
| `Assets/Resources/Prefabs/UI/Recruit/RecruitPage.prefab` | 페이지 프리팹 |
| `Assets/Resources/Prefabs/UI/Recruit/RecruitItem.prefab` | 아이템 프리팹 |

---

## 영입 흐름

```
RecruitPage.OnOpened()
    └── _recruitPawns가 없으면 GenerateRecruitList()
            ├── PawnTable.GetAll() → 전체 PawnData 목록
            └── 랜덤 3개 선택 → new DPawn(data) (CodeName 자동 할당)
    └── RefreshItems()
            ├── 기존 RecruitItem 전부 Destroy
            └── _recruitPawns 각각 → RecruitItem 인스턴스 생성 → SetData(pawn)

RecruitItem.OnClickRecruit()
    ├── DeckManager.Instance.AddPawn(_pawn)
    └── OnRecruited 이벤트 발생 → RecruitPage.OnItemRecruited()
            ├── _recruitPawns에서 제거
            ├── _activeItems에서 제거
            └── gameObject Destroy
```

---

## RecruitPage 주요 멤버

| 멤버 | 설명 |
|------|------|
| `_activeItems` | 현재 표시 중인 RecruitItem 리스트 (UI 전용) |
| `RefreshItems()` | 기존 아이템 제거 후 `LobbyManager.RecruitPawns` 기반으로 재생성 |
| `RefreshGoods()` | 보유 골드 텍스트 갱신 |
| `Refresh()` | 외부에서 목록 강제 갱신 → `LobbyManager.GenerateRecruitList()` 호출 |

> 목록 데이터(`RecruitPawns`, `RecruitSlotCount`)는 `LobbyManager`가 보관.

## LobbyManager 영입 관련 API

| 메서드/프로퍼티 | 설명 |
|----------------|------|
| `RecruitPawns` (IReadOnlyList) | 현재 영입 후보 목록 |
| `GenerateRecruitList()` | 전체 PawnData에서 랜덤 3개 뽑아 DPawn 생성 |
| `RemoveRecruitPawn(pawn)` | 영입 완료 후 후보 목록에서 제거 |

---

## RecruitItem 주요 멤버

| 멤버 | 설명 |
|------|------|
| `SetData(DPawn pawn)` | CodeName, ClassType, Price 표시 |
| `OnClickRecruit()` | DeckManager.AddPawn + OnRecruited 이벤트 |
| `OnRecruited` (`Action<RecruitItem, DPawn>`) | 영입 완료 시 발생 |

---

## 후보 목록 갱신 시점

| 시점 | 갱신 여부 |
|------|----------|
| 페이지 최초 오픈 | O (목록이 없을 때만) |
| 페이지 닫고 재오픈 | X (기존 목록 유지) |
| Reroll 버튼 클릭 | O |
| Stage 복귀 | O (`Refresh()` 외부 호출 필요 — 미연결) |
