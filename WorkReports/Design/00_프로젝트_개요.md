# 프로젝트 개요

## 게임 장르 및 컨셉

턴제 전술 카드 게임. 타일맵 위에서 폰(Pawn)을 이동시키고 카드를 사용해 몬스터를 처치하는 방식의 전략 RPG.

---

## 기술 스택

| 항목 | 내용 |
|------|------|
| 엔진 | Unity |
| 언어 | C# |
| 데이터 직렬화 | Google Protocol Buffer (Protobuf) |
| 데이터 저장 | `.bytes` 파일 (Resources 폴더) |
| 에셋 관리 | Resources.Load + AssetPathAttribute |

## 싱글톤 패턴

### MonoSingleton\<T\> (MonoBehaviour 기반)
- `Assets/Scripts/Core/MonoSingleton.cs`
- Instance 최초 접근 시 자동 생성
- `[AssetPath("경로")]` 있으면 Resources에서 프리팹 로드 후 Instantiate
- 없으면 `new GameObject()` + `AddComponent<T>()`
- `ManagerRoot` GameObject(DontDestroyOnLoad)의 자식으로 배치
- **사용**: StageManager, TurnManager, DeckManager, TileManager, UIManager, PlayerManager 등

### Singleton\<T\> (순수 C# 클래스)
- `Assets/Scripts/Core/Singleton.cs`
- `new T()`로 생성 (MonoBehaviour 아님)
- **사용**: SceneController, DataManager
