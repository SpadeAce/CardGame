## 좌표 시스템

- 그리드: `Vector2Int(x, y)` → 월드 좌표 X, Z 축
- 타일 간격: `tileSize` (기본 1.05f)
- 4방향: North(0,1) / East(1,0) / South(0,-1) / West(-1,0)

---

## 타일 종류 (TileType)

| 값 | 이름 | 이동 가능 | 설명 |
|----|------|----------|------|
| Empty | 빈 공간 | ❌ | 타일 없음 |
| Ground | 지면 | ✅ | 일반 지형 |
| Dirt | 흙 | ✅ | 흙 지형 |
| Water | 물 | ❌ | 이동 불가 |
| Blocked | 막힘 | ❌ | 차단된 타일 |

---

## 타일 상태 (TileState)

| 상태 | 설명 |
|------|------|
| Normal | 기본 |
| Selected | 클릭 선택됨 (노란 테두리 등) |
| Highlighted | 이동 가능 범위 표시 (파란 하이라이트) |
| Disabled | 비활성화 |

추가 시각화: `SetTarget(bool)` — 카드 사거리 표시 (빨간 하이라이트)

---

## EdgeTile (타일 간 경계)

- 인접한 두 타일 사이의 경계에 배치
- 키: `EdgeTileKey(Vector2Int posA, Vector2Int posB)`
- 용도: 벽(`TileWall`) 배치 → 경로탐색에서 통과 불가로 처리
- 방향: North / East / South / West

---

## TileMapPreset (ScriptableObject)

- **위치**: `Resources/Tile/Preset/`
- **필드**:
  ```
  int width, height     : 맵 크기
  float tileSize        : 타일 간격 (기본 1.05f)
  List<TilePresetData> tiles : 각 타일 설정
  List<EdgePresetData> edges : 경계(벽) 설정
  ```
- **TilePresetData 필드**:
  ```
  Vector2Int position   : 그리드 위치
  TileType tileType     : 타일 종류
  SpawnPointType spawnPoint : 스폰 포인트 종류 (Player/Enemy)
  int spawnId           : Pawn ID 또는 Monster ID
  int entityId          : EntityCatalog ID (타일 오브젝트)
  ```
- **EdgePresetData 필드**:
  ```
  Vector2Int posA, posB : 두 타일 위치
  int entityId          : 배치할 엔티티 (벽 등)
  ```

## 경로탐색 알고리즘

### GetReachableTiles() — 도달 가능 범위
- **알고리즘**: BFS (너비 우선 탐색)
- **이동 규칙**:
  - Water / Blocked 타일 진입 불가
  - EdgeTile에 TileWall이 있으면 통과 불가
  - 같은 진영 유닛 타일 통과 가능 (단, 정지 불가)
  - 다른 진영 유닛 타일 통과/진입 불가

### GetPath() — 경로 계산
- **알고리즘**: A* (맨해튼 거리 휴리스틱)
- **이동력 제한**: movement 이내 경로만 반환

### SmoothPath() — 경로 스무딩
- **알고리즘**: Greedy 직선 시야 검사
- 웨이포인트를 최소화하여 자연스러운 이동 경로 생성
- `HasLineOfSight()` — Bresenham 직선 알고리즘 + 코너 커팅 방지

## 맵 생성 흐름

```
StageManager.LoadStage(preset)
    └─▶ TileManager.GenerateTileMapFromPreset(preset)
              ├─ 각 TilePresetData → SquareTile 프리팹 인스턴스화
              ├─ 각 EdgePresetData → EdgeTile + TileWall 배치
              └─ SpawnPoint 타일 반환
    └─▶ PlayerSpawn 위치 → Actor(DPawn) 스폰
    └─▶ EnemySpawn 위치 → Actor(DMonster) 스폰
```

---

## EntityCatalog

- **위치**: `Resources/Data/EntityCatalog.asset`
- **용도**: entityId → 프리팹 경로 매핑
- **사용**: TileMapPreset의 entityId로 TileWall 등 배치
