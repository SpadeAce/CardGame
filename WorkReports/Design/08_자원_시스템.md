# 자원 시스템

## 구현 상태: 기본 완성 (공용 ActingPower + Pawn 개별 Ammo + 로비 Gold 구현 완료)

| 파일 | 역할 |
|------|------|
| `Assets/Scripts/Logic/Manager/TurnManager.cs` | SharedActingPower 관리 |

## 자원 종류

### 1. SharedActingPower (공용 행동력)

카드의 `EnergyCost`를 소모하는 **팀 공유 자원**.

| 항목 | 내용 |
|------|------|
| 관리 위치 | `TurnManager.SharedActingPower` |
| 기본값 | 5 (`SharedActingPowerPerTurn`) |
| 리셋 시점 | Pawn 턴 시작 시 (`StartPawnTurn()`) |
| 초과 허용 | 카드 효과로 5 초과 가능 (다음 턴에 기본값으로 리셋) |

**주요 API (TurnManager)**
| 메서드/이벤트 | 설명 |
|--------------|------|
| `SharedActingPower` (int) | 현재 공용 행동력 |
| `SharedActingPowerPerTurn` (const = 5) | 턴 당 기본 지급량 |
| `HasEnoughSharedActingPower(cost)` | 공용 행동력 충분 여부 |
| `ConsumeSharedActingPower(amount)` | 공용 행동력 소모 (0 미만 방지) |
| `AddSharedActingPower(amount)` | 공용 행동력 추가 (한도 없음) |
| `onSharedActingPowerChanged` | 변경 시 발생하는 이벤트 |

---

### 2. Ammo (Pawn 개별 탄약)

카드의 `AmmoCost`를 소모하는 **각 Pawn 보유 자원**.

| 항목 | 내용 |
|------|------|
| 관리 위치 | `DPawn.Ammo` |
| 초기값 | `PawnData.ActingPower` (데이터 필드 재활용) |
| 매 턴 자동 회복 | 없음 |
| 회복 방법 | `Reload` 카드 효과 사용 시 기본값으로 전체 회복 |

**주요 API (DPawn)**
| 메서드 | 설명 |
|--------|------|
| `Ammo` (int) | 현재 탄약 수치 |
| `HasEnoughAmmo(cost)` | 탄약 충분 여부 |
| `ConsumeAmmo(amount)` | 탄약 소모 (0 미만 방지) |
| `RestoreAmmo()` | 탄약 전체 회복 (`PawnData.ActingPower` 값으로) |
| `AddAmmo(amount)` | 탄약 증가 |

---

---

### 3. Gold (로비 재화)

상점 카드 구매 및 Pawn 영입에 사용되는 **로비 전용 재화**.

| 항목 | 내용 |
|------|------|
| 관리 위치 | `PlayerManager._gold` |
| 초기값 | 1000 |
| 소모 시점 | 상점 카드 구매 / Pawn 영입 |
| 획득 방법 | 스테이지 클리어 시 (몬스터 처치 골드 합계 + 클리어 보너스 100) |

**주요 API (PlayerManager)**
| 메서드/프로퍼티 | 설명 |
|----------------|------|
| `Gold` (int) | 현재 보유 골드 |
| `SpendGold(amount)` | 골드 차감. 잔액 부족 시 false 반환 |
| `AddGold(amount)` | 골드 추가 |

**구매 가격 기준**
| 구매 대상 | 가격 |
|----------|------|
| 상점 카드 | `CardData.Price` (데이터 기반) |
| Pawn 영입 | 100 (임시 고정값, `RecruitItem.RecruitPrice`) |

## 카드 비용 구조

`CardData`는 두 종류의 비용 필드를 가진다.

| 필드 | 타입 | 소모 자원 |
|------|------|-----------|
| `EnergyCost` | int | SharedActingPower (공용) |
| `AmmoCost` | int | Pawn.Ammo (개별) |

카드 사용 가능 조건: **두 자원 모두** 충족해야 사용 가능.

```
CanAffordCard(card, pawn)
    = HasEnoughSharedActingPower(card.EnergyCost)
    && pawn.HasEnoughAmmo(card.AmmoCost)
```

---

## 카드 효과 연동

| 효과 | 동작 |
|------|------|
| `RestoreAction` | `TurnManager.AddSharedActingPower(value)` — 공용 행동력 회복 |
| `Reload` | `targetPawn.RestoreAmmo()` — Pawn 탄약 기본값으로 전체 회복 (value 무시) |

---

## 자원 갱신 흐름

```
Pawn 턴 시작 (TurnManager.StartPawnTurn)
    → SharedActingPower = 5
    → onSharedActingPowerChanged 발생
        → StagePage.SetCardList()  : 글로벌 핸드 카드 활성화 갱신
        → StagePage.SetPawnCardList(): Pawn 핸드 카드 활성화 갱신

카드 사용
    → SpendCardCost(card, pawn)
        → ConsumeSharedActingPower(EnergyCost)
            → onSharedActingPowerChanged → 카드 UI 갱신
        → pawn.ConsumeAmmo(AmmoCost)
            → onStatsChanged → HUD_Actor Ammo 표시 갱신 + 카드 UI 갱신

RestoreAction 카드
    → AddSharedActingPower(value)
        → onSharedActingPowerChanged → 카드 UI 갱신

Reload 카드
    → pawn.RestoreAmmo()
        → onStatsChanged → HUD_Actor Ammo 표시 갱신 + 카드 UI 갱신
```
